We Love LOOPS!

For Loops

Use a for loop when you want to run a loop a certain number of times.
You track how many times your loop has been executed by a counter variable

  • Utilize the for keyword
  • Within parentheses we define constraints: how many times will this loop run?
  • Assign a variable that indicates where to start: let i = 0
  • We use our variable as an operand then a comparison operator followed by a second operand
  • For example: i < array.length or i < 5
  • This is essentially a conditional value to determine when to break the loop
  • We use the increment operator or the decrement operator to increase/decrease the value of our assigned variable
  • For example: i++ or i--
  • We open our code block with curly brackets
  • Our statements, which are instructions go within these curly brackets

While Loops

Use a while loop when you don't know how many times you need to loop - maybe even zero times.

  • Will keep running until your condition becomes false
  • You will end up in an infinite loop if you do not include a way to make your condition evaluate to false
  • Utilize the while keyword
  • Only takes one condition in the parentheses
  • For example: (userAttempts)
  • Open code block with curly brackets...

Do...While Loops

Use a 'do...while' loop when you know you want to run the loop once, but don't know how many more times.

  • These will run once regardless of the condition
  • Should be utilized in situations where the loop body needs to be executed at least once: such as menu driven programs
  • Utilize the do keyword
  • Open code block with curly brackets
  • Statements (instructions) go within these brackets
  • After closing curly bracket utilize while keyword
  • Condition goes within parentheses
  • Loop ends when this condition evaluates to false
