CLASS 02

Class Attribute

  • Used to select multiple elements with the same class
  • Can be utilized by either CSS or JavaScript to target elements
  • Accessed in CSS via the . symbol

ID Attribute

  • Each ID is unique and can only be used once per page
  • Accessed in CSS via the # symbol
